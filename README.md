# monkey-face
## Cara download program ini  
Klik "code" yang berwarna hijau, dan klik download dalam format zip, setelah itu ekstrak pada folder lokal.  
## Tools yang dibutuhkan untuk menjalankan program ini  
### Perangkat Lunak  
- VSCode (text editor)
- Python versi 3.13.0 (recommended)
### Library Python (install via terminal VSCode)
- pip install opencv-python
- pip install mediapipe
- pip install numpy
